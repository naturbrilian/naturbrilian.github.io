Tips & Tricks how to lock Facebook account (Not working at the moment)

Hello, it feels like I haven’t written an article on this personal blog for a long time, because I’ve been busy playing Princess Connect Re:Dive and have been busy in RL. So I took the time to write an article about a tutorial again, so yesterday on one of the Discord servers called Yurasubs server, one of the admins asked me, how to lock a Facebook account like this

So previously I have made a tutorial, but unfortunately I did not upload the tutorial video on my usual account, but on a real life account, because I am sure you understand, the previous tutorial has been shared by one of the Facebook account users, only the post was drowned, let’s start the steps, okay.

Sign in to your account

Make sure the account you are using has never been locked before, the first step is to go to the settings section, then you go to the Language tab, this is the most important (because the current author’s display language is Javanese)

Language

Of course, for this feature you have to change the language to Burmese/Myanmar, because this setting or feature is not available if you use Indonesian or Javanese, let’s change the interface of your account. Then if the interface change to Myanmar, dont forget to use a translator tools.

Change the display to mbasic if you are still confused

If you are confused about finding the “locked profile” setting, you can change the Facebook display to mbasic, just type it in the browser bar as usual, then go to settings > scroll down (Private section) until you see a lock image like this (the image on the side/above), then press “close your profile”

Done

If so, the display will look like this, meaning it was successful. Don’t forget to change the display language back to your preferred language.
